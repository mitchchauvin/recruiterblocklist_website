Welcome developers!

Use our API in your software or mobile app. Our API is easy to use and open-source (GPL). Our database of recruiters is community supported to stay ahead of the ever changing recruiter landscape!

Our API is so simple, we bet can learn it in 5 minutes! Scroll down to get started!

You can also contribute to our open source website here!

Get a list of recruiter domains in JSON format

Get the complete list of recruiter domain names in JSON format.

Method:GET
URL:http://www.recruiterblocklist/api.php/recruiter_domains/get_all_as_json
Response:JSON encoded data. Ex. {"domain":"example1.com", ...}


Get a list of recruiter domains in TEXT format

Get the complete list of recruiter domain names in TEXT format. The list is composed of one domain per line.

Method:GET
URL:http://www.recruiterblocklist/api.php/recruiter_domains/get_all_as_text
Response:List of recruiter domains, one domain per line, in text format.


Get a list of recruiter domains in CSV format

Get the complete list of recruiter domain names in CSV format. The list is composed of one domain per line.

Method:GET
URL:http://www.recruiterblocklist/api.php/recruiter_domains/get_all_as_text
Response:List of recruiter domains, one domain per line, in CSV format.



Get a list of recruiter domains in XML format

Get the complete list of recruiter domain names in Gmail mail filter XML format. This format can be imported directly into Gmail using the mail filter tool to block unwanted recruiter mail!

Method:GET
URL:http://www.recruiterblocklist/api.php/recruiter_domains/get_all_as_xml
Response:Gmail mail filter format (XML).



Check if a recruiting company's domain is on the block list

Method:POST
URL:http://www.recruiterblocklist/api.php/recruiter_domains/is_in_list
Arguments:domain as string.
Response:JSON: {"success":[true,false]}.



Get a complete list of recruiters in JSON format

Method:GET
URL:http://www.recruiterblocklist/api.php/recruiters/get_all_as_json
Response:JSON: {"success":[true,false], "result": Object containing all recruiter info}.
